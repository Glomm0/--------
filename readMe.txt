Параметры работы алгоритма можно изменять в Файле main.py после проверки условия if __name__=="__main__"


Уравнения для проверки записываются в файл examples.txt, разделяясь ",\n" (через запятую и переход на новую строку) В файл можно написать не более 3 функций из-за ограничений библиотеки по отрисовке графиков

Так же можно вызвать функцию generate_into_file, изменив количество переменных уравнений в соответствующем файле (функция в самом низу)

Графики после работы функции рисуются для каждого уравнения и показвают максимальное кол-во выполнимых скобок на определенном шаге алгоритма
