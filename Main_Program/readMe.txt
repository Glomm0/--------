Параметры задаются в файле parameters.txt:
    a1 - влияние лучшей позиция частицы
    a2 - влияние лучшей позиция роя
    w - инерция

    все параметры должны быть больше нуля и значением меньше 5.

Уравнение для проверки записывается в файл examples.txt, ограничение в 1 уравнение одновременно!!

Графики после работы функции рисуются для каждого запуска и показвают максимальное кол-во выполнимых скобок на определенном шаге алгоритма

При запуске программы можно выбрать 2 алгоритма решения:
    Бинарный - используются побитовые операторы для работы алгоритма (Лучше ищет решение, если существует решение всех скобок)
    Вероятностный - используется подход, что скорость является вероятностью той или иной переменной быть 1 или 0 (В среднем показывает себя лучше первого)

После отрисовки графика, можно изменить параметры в файле и ввести 1 в консоль, для повторного запуска, в этом случае на гррафике выведется результат работы нового запуска вместе со старым для сравнения
